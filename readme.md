# 如何安裝專案到自己的電腦?
1. 滑鼠右鍵開啟git bash

2. 到Code資料夾下，請先將Code資料夾下的ITS刪除後

3. 輸入

`git clone https://github.com/tyson3822/SE-IssusTrackingSystem.git ITS` //下載專案，資料夾命名為ITS

`cd ITS` //進入ITS資料夾

`git checkout develop` //從網路上抓下develop這個branch

4. 完成

# 安裝套件
1. 進入vagrant，到專案目錄下(~/Code/its/)
2. 執行指令
`composer update`
